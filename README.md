# 北航课程伴侣 APP

一款专为北航学生设计的课程管理应用，帮助同学们清晰管理课表与任务，轻松组建学习小组。

## 功能特性

### 基本功能

1. **课程表管理**
   - 支持课程信息的录入、编辑与查询
   - 提供可视化的课程表展示
   - 课程信息包括：课程名称、上课时间、地点、任课教师等
   - 支持设置上课提醒

2. **作业提醒**
   - 支持创建作业和实验任务
   - 记录任务详情、所属课程及截止时间
   - 提供灵活的提醒功能
   - 帮助用户跟踪任务进度（未开始、进行中、已完成、已逾期）

3. **学习小组组局**
   - 支持用户根据课程或主题创建学习小组
   - 能够邀请好友或允许其他同学加入
   - 提供小组内的基础交流空间

### 选做功能

1. **AI 笔记助手**
   - 支持用户上传课堂录音或笔记图片
   - 自动生成结构清晰、重点突出的知识提纲
   - 助力高效复习

2. **作业辅导提示**
   - 当用户在作业上遇到困难时，可输入问题
   - 由大模型提供解题思路提示与相关知识点的梳理
   - 启发用户自主完成

3. **个性化学习分析**
   - 根据用户的课程表、作业完成情况与小组活跃度
   - 利用数据分析生成学习报告与个性化学习建议

## 技术栈

- **开发语言**: Kotlin
- **UI框架**: Jetpack Compose
- **数据库**: Room (本地) + MySQL (服务器端)
- **架构**: MVVM
- **依赖注入**: 手动依赖注入
- **导航**: Navigation Compose
- **异步处理**: Kotlin Coroutines
- **后台任务**: WorkManager

## 数据库设计

项目包含完整的MySQL数据库设计，支持以下表结构：

- `users` - 用户表
- `courses` - 课程表
- `assignments` - 作业/实验任务表
- `study_groups` - 学习小组表
- `group_members` - 小组成员表
- `group_messages` - 小组消息表
- `notes` - 笔记表（选做功能）
- `assignment_help` - 作业辅导记录表（选做功能）
- `learning_analytics` - 学习分析记录表（选做功能）
- `notifications` - 通知提醒表

详细表结构请参考 `database_schema.sql` 文件。

## 安装与运行

1. 克隆项目到本地
2. 使用 Android Studio 打开项目
3. 同步 Gradle 依赖
4. 运行应用

## 使用说明

### 课程表管理

1. 进入"课程表"页面
2. 点击右下角"+"按钮添加课程
3. 填写课程信息（课程名称必填）
4. 选择上课星期和时间
5. 设置提醒（可选）
6. 保存课程

### 作业管理

1. 进入"作业"页面
2. 点击右下角"+"按钮添加作业
3. 填写作业信息
4. 设置截止时间和优先级
5. 保存作业
6. 可通过状态筛选查看不同状态的作业

### 学习小组

1. 进入"学习小组"页面
2. 点击右下角"+"按钮创建小组
3. 填写小组信息
4. 设置是否公开
5. 创建小组后可以邀请成员加入

### AI功能（选做）

1. 进入"我的"页面
2. 点击"AI笔记助手"上传录音或图片
3. 点击"作业辅导提示"输入问题获取帮助
4. 点击"学习分析"查看学习报告

## 注意事项

1. 当前版本使用本地Room数据库，用户ID默认为1
2. AI功能需要接入实际的AI API（当前为模拟实现）
3. 通知功能需要用户授权通知权限
4. 文件上传功能需要实现文件选择器
